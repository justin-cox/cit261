Week 8 Notes

Transform Property

  • Lets you lets you translate, rotate, scale, and/or skew any element on the page.

    • Translation

  • Allows you to move elements left, right, up, or down

    • Scaling

  • Scales an element by the defined factors horizontally then vertically.

    • Rotation

  • Rotates an element around the point of origin by a specified angle value.

    • Skew

  • Specifies a skew along the x and y axes

    • transition-property

  • Defines the CSS properties of the element that should be transitioned, with all for all properties being the default.
  • You can have multiple transitions in one call.

    • transition-duration

  • Sets how long the transition will take: the duration of time it takes to go from the default state to the transitioned state.

    • Animations

  • Allow for element animation over time through the use of keyframes
  • You can also change direction, number of iterations, add delays, and many more.

    • Keyframes

  • The percentage value is what is important. Not the order that they are written

    • animation-name

  • Allows you to assign a created name to an animation

    Canvas

  • Don't set the width and height of a canvas element using css. Set it to the element itself.
  • You can add borders.
  • Use WebGL for 3D
  • You can save canvas drawings using the toDataURL method.
  • You can also manipulate video, display text, etc...

    • SVG

  • Stands for scalable vector graphics
  • Allows for use of vector graphics using XML
  • Much easier to use SVG than canvas for certain things.

    • You can drag and drop files with the file api.
